# Workbook UI Kit

Interactive click-through prototype of the Sovereign Stillness 30-day digital workbook. Built as a portrait-format digital reader (8.5×11 pages).

## Pages Modelled
- Cover
- Table of Contents
- Chapter Opener (dark)
- Body Text page
- Quote page (large)
- Two-Column layout
- CTA / Unlock page
- Thank You / closing

## Usage
Open `index.html` in any browser. Navigate with arrow keys or click the page edges.
